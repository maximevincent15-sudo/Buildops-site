# BuildOps

SaaS de gestion pour les entreprises de maintenance en sécurité incendie (3 à 20 techniciens).

**Équipements couverts :** extincteurs, RIA, désenfumage, SSI.

**Promesse :** planifier les interventions, générer des rapports automatiques, rester conforme sans effort.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS (design system custom via CSS variables)
- React Router v6
- Zustand (state management)
- Supabase (auth + PostgreSQL + Storage, RLS multi-tenant)
- react-hook-form + zod
- @react-pdf/renderer

## Modules prioritaires

1. **Planning des interventions** — créer, assigner, suivre
2. **Rapport d'intervention mobile** — checklist, photos, signature, PDF auto
3. **Fiche client** — sites, équipements, historique
